# Academic CV — LaTeX Template

A clean, minimal LaTeX CV template for academic use, structured after a Georgia Tech faculty CV style.

## Structure

```
.
├── main.tex   # Main CV source file
├── main.pdf   # Compiled output
└── README.md
```

## Sections

- Research Area
- Professional Appointments
- Education
- Honors and Awards
- Research Funding
- Publications (Book Chapters, Journals, Conferences)
- Workshops and Tutorials
- Keynote and Invited Talks
- Teaching Experience
- Student Advising
- Professional Services

## Dependencies

Standard packages only — no custom `.sty` files needed:
`geometry`, `titlesec`, `tabularx`, `hyperref`, `enumitem`, `parskip`, `microtype`
